# Daroku

タイピングソフトの結果を記録・管理するmacOSアプリケーションです。

## 機能

### タイピングソフト管理
- 複数のタイピングソフト（寿司打、e-typingなど）をカテゴリ別に管理
- ソフトごとにスコアの単位（円、点、WPMなど）を設定可能

### 記録機能
- **手入力**: スコア、正確キー数、ミスタイプ数、平均キータイプ速度を記録
- **画像認識**: Vision frameworkによるOCRで結果画面からテキストを抽出
- **カスタムフィールド**: ソフト固有のデータを自由に追加可能
- **メモ**: 各記録にメモを追加可能

### 表示機能
- **表形式**: 記録を一覧表示、ソート機能付き
- **折れ線グラフ**: 時系列での推移を視覚化
  - スコア、正確キー数、ミスタイプ数、平均速度を表示
  - 複数指標の同時表示に対応
  - 統計サマリー（平均、最高、最低）表示

## スクリーンショット

```
+------------------+----------------------------------------+
|  サイドバー       |  メインコンテンツ                        |
|                  |                                        |
|  [+ 追加]        |  寿司打                                 |
|                  |  単位: 円                               |
|  - 寿司打        |  [表] [グラフ]                          |
|  - e-typing      |                                        |
|                  |  日付 | スコア | 正確性 | 速度          |
|                  |  12/30| 4620円 | 443回  | 3.5回/秒      |
+------------------+----------------------------------------+
```

## 動作環境

- macOS 26.0以降
- Swift 6.0

## ビルド方法

このプロジェクトは[Tuist](https://tuist.io/)を使用しています。

```bash
# プロジェクトの生成
tuist generate

# ビルド
tuist build
```

## 技術スタック

- **UI**: SwiftUI
- **グラフ**: Swift Charts
- **画像認識**: Vision framework
- **データ永続化**: Core Data
- **ビルドツール**: Tuist
